# Plan: Dashboard (인덱스 메인화면) 완전 개편

## 1. 개요

### 배경
- 현재 대시보드는 이전 임시화면에서 1차 개선된 상태
- 기본 통계 카드 6개 + 도넛 차트 3개 + Top10 바차트 + DC별 스택차트 + 멀티캐스트/PTP 테이블로 구성
- 시스템에 많은 메뉴와 데이터가 추가되었으나 대시보드에 미반영 항목 존재
- API에서 이미 반환하는 expiring_circuits 등 미사용 데이터 활용 필요

### 목표
- 네트워크 통합운영시스템의 **모든 핵심 정보를 한 페이지**에서 확인 가능한 종합 대시보드 구현
- 실시간 장비 상태(멀티캐스트, PTP)와 DB 기반 통계를 효과적으로 결합
- 직관적인 시각화와 빠른 네비게이션 제공

### 현재 대시보드 (index.html) 구성
1. **1행**: 요약 카드 6개 (총 회선, 회원사, 시세상품, 요금기준, 멀티캐스트 상태, PTP 상태)
2. **2행**: 도넛 차트 3개 (통신사별, 환경별, 용도별) + Top 10 회원사 바차트
3. **3행**: DC코드별 스택 바 차트(col-5) + 멀티캐스트 상세 테이블(col-7)
4. **3.5행**: PTP 상세 테이블 (col-12 전체 폭)
5. **4행**: 빠른 이동 Quick Links 5개

### 사용 가능한 API 데이터 (FastAPI /dashboard)
- summary: total_circuits, total_members, total_products, total_fee_items, expiring_soon
- circuits_by_provider, circuits_by_env, circuits_by_usage, circuits_by_dc
- top_members, **expiring_circuits** (90일 이내 만료 - 현재 미사용)
- 멀티캐스트: /multicast/init (실시간, ~25초 타임아웃)
- PTP: /information/init?sub_menu=info_ptp (실시간, ~20초 타임아웃)

### DB 현황
- circuit: 459건 (ORD 207, MPR 201, MGT 51 / PR 232, DR 107, TS 69, MG 51 / LGU 338, KT 121)
- 만료 임박 회선: 현재 0건 (expiry_date 미입력)

#### 신규 추가 세부

**3행 우측 - 대역폭별 회선 분포** (신규)
- 가로 바 차트 (ECharts)
- ORD 회선 기준 대역폭별(10M, 50M, 100M, 150M, 200M, 450M) 분포

- 기존 3행+3.5행을 4행 하나로 통합
- 멀티캐스트: col-md-7, PTP: col-md-5
- 테이블 max-height 동일하게 맞춤

**5행 - 만료 임박 + 최근 등록** (신규)
- 좌측: 만료 임박 회선 (API에 이미 `expiring_circuits` 존재, 현재 미사용)

- 우측: 최근 등록/수정 회선 최근 10건

**6행 - Quick Links 확장**
